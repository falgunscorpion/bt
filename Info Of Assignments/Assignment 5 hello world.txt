# Theory

A smart contract is a self-executing program stored on the blockchain. Ethereum smart contracts are generally written in Solidity.

In this assignment, a HelloWorld smart contract is compiled using Solidity compiler (`solc`) and deployed on an Ethereum test network using Web3.py.

The deployment process includes:

* Connecting to Ethereum node using RPC URL
* Compiling Solidity code
* Generating ABI and bytecode
* Creating deployment transaction
* Signing transaction using private key
* Broadcasting transaction to blockchain

The deployed contract receives a unique contract address on the Ethereum blockchain.

# Explanation of Code

The program deploys a HelloWorld smart contract on an Ethereum test network using Python and Web3.py. 

First, environment variables such as RPC URL, private key, and account address are loaded securely using dotenv. The program then connects to the Ethereum testnet using Web3 HTTP provider.

The Solidity source code (`HelloWorld.sol`) is loaded and compiled using Solidity compiler version 0.8.20. Compilation generates ABI and bytecode required for deployment.

Next, a contract object is created using ABI and bytecode. A deployment transaction is built with parameters such as sender address, nonce, gas limit, gas price, and chain ID.

The transaction is digitally signed using the private key and sent to the Ethereum network. After mining confirmation, the deployed contract address is obtained and stored in a JSON file along with ABI.
